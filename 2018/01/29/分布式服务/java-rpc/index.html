什么是 rpc

RPC（Remote Procedure Call）—远程过程调用，它是一种通过网络从远程计算机程序上请求服务，而不需要了解底层网络技术的协议。

简单的说就是在一台机器上使用另一台机器上的一个方法。

用 java 的 socket 以及 bio 实现一个最简单的 rpc。

首先，要有一个接口，服务提供者实现该接口，并将该实现暴露，提供给消费者使用。服务消费者可以通过 rpc 得到服务提供者提供的接口实现，来方法提供者提供的方法。
